# YTS Clone

## What is being used?
  - Axios for apis handling
  - searching movies
  - useContext for state management
  - Tailwindcss, typescript

## Features
  - Fetch movies 
  - View movie details 
  - View movie trailers
  - Search movies 
  - Theme Switch
  - Pagination
  - Coming Soon Page
